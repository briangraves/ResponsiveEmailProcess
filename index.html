Responsive Email Process

  • Discovery
  • Content Strategy
  • Design
  • Development
  • Execution

Discovery

  • Project Analysis

    Determine project requirements for content, design, technical, and organizational perspective

  • Goal Setting

    Determine goals for success and analytics to be tested and measured

Content Strategy

  • Content Inventory

    List all of the content with quantitative information. published date, type of content, etc.

  • Content Audit

    Analyze current content & sends. Determine what's working, what's not, and if anything content types need to be rewritten, redesigned, added, or removed. Qualitative analysis.

  • Content Packaging & Priority

    Create content packages and priority guides are created to align content with marketing goals and create a sustainable format for future emails.

Design

  • User Experience Design

    High level content packages & priority guides and low-fi graphics are used to determine how content will reflow across screen sizes.

  • Visual Design

    Visual comps are created that align with the Content Strategy and Responsive Wireframes as well as Brand Guidelines

Development

  • HTML & CSS Build

    Responsive Wireframes and Visual comps are used to create a Template build as well as any necessary Content Packages

  • Engineering

    Any necessary back-end integration is tied to the template implementation

Execution

  • Training

    Any necessary training

  • Implementation

    Daily Sends

  • Analytics

    Measurement of analytics and determination of success
